# 2D_Terrain_Generation  
/!\ currently in a bad state  

## USAGE  
./PerlinNoise cellSize  
  
Infinite 2D terrain generation made with CSFML using perlin noise  
You can press the [LEFT] or [RIGHT] arrows to side scroll the generation.
